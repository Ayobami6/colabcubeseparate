## Packages to be installed in backend

### Change to the backend directory

```
cd frontend

```

### then run

```
  npm i bcryptjs

```

#### Get JWT_SECRET

```
   openssl rand -hex 64

```

### then run this command

```
npm install

```

```

npm install express nodemon cors mongoose dotenv

```
